About the device

What TinyML Smart Plug is for

The TinyML Smart Plug is a complementary protection device for residential use. It is designed to monitor voltage, current, temperature, and waveform behavior so it can warn or trip on dangerous conditions that normal breakers may not explain clearly to the user, especially progressive fault conditions.

It works as an additional safety layer. It does not replace required electrical protection devices.

How it thinks

The device uses a hybrid method. Arc faults are detected by an AI model using waveform features, while overload, temperature, and voltage protection use fixed safety rules for fast decisions.

What you see on the dashboard

Status banner

This is the fastest summary of the latest device condition. If live data stops updating for 15 seconds, the dashboard changes to Disconnected so stale values are not mistaken as current.

System overview

This area summarizes overall health, connectivity, power condition, active protection state, and the most recent recorded event.

Live electrical data

Voltage, current, apparent power, and temperature come from the most recent live packet. Hover the small ? in each card for a quick explanation.

Waveform data

These cards show the feature values used for waveform analysis and AI-based arc evaluation. When live data becomes stale, these values also switch to disconnected state.

History

History shows important device states over time. Use the date filter to narrow what you see, and use download when you want a CSV export.

Guide access

The question mark at the top-right of the dashboard opens this full guide so the main dashboard stays cleaner and less cluttered for daily use.

Alerts and status meanings

Monitoring

The device is connected and no active trip condition is being reported from the latest packet.

Disconnected

The dashboard has not received a fresh live packet for at least 15 seconds.

Unplugged

The smart plug is still reachable, but mains voltage is absent or near zero.

Critical fault

The device is reporting a trip or a severe condition such as arc fault, short circuit, or socket heating.

Enable browser alerts if you want system notifications to appear on your phone or computer when important conditions happen.

Protection rules

Arc Fault

Trips when the AI model detects a high probability of arcing from waveform features.

Overload

Alarms when current reaches 10 A or above.

Socket Heating

Trips when socket temperature reaches 70 °C or higher.

Short Circuit

Trips when measured current goes above 20 A.

Undervoltage

Trips when voltage drops below 200 V.

Overvoltage

Trips when voltage rises above 250 V.

Waveform feature guide

Cycle NMSE

Compares one electrical cycle against the next. Larger changes can suggest abnormal repeatability.

Zero-Cross Variance

Measures how much the zero-cross timing varies. Higher irregularity can support arc-related detection.

Zero-Cross Dwell Ratio

Shows how much of the waveform lingers near the zero region, which can help separate fault behavior from normal loads.

Pulse Count and Peak Fluctuation

These describe how spiky and unstable the waveform looks across a cycle.

HF Band Energy and Entropy

These values help describe high-frequency noise and randomness, both of which are useful when looking for arc behavior.

THD

Total Harmonic Distortion estimates how much the waveform departs from a cleaner sinusoidal shape.

Advanced tools in the web app

History and CSV export

Use the history table for fault review, event review, and simple record keeping. Download exports the current filtered view as CSV.

Data Logger

The Data Logger helps with model training and controlled data gathering. You can set duration, indicate load type, apply a label override, then download one session or compile all sessions.

Plot Viewer

The Plot Viewer opens session data for deeper inspection. It lets you review feature curves, move through time, smooth data, normalize values, and inspect recorded events more clearly.

OTA Updates

OTA lets you publish a target firmware version and firmware link so the device can pull a newer build. Keep this collapsed unless you are actively maintaining firmware.

Installation and daily use

Install the PWA

Add the website to your home screen or install it from your browser so it behaves more like an app and opens faster in daily use.

Keep the dashboard simple

The main dashboard is intentionally compact for consumers. Quick hints stay inside hover help, while the full explanations live here in the guide.

Best practice

If the device shows a protection state, stop using the connected appliance until the cause is checked. A smart plug can warn early, but safe inspection still matters.
